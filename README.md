# Sunu Santé - Application de Prise de RDV (Dakar)

Ceci est un prototype fonctionnel de l'application **Sunu Santé**, conçue pour faciliter la prise de rendez-vous dans les structures sanitaires de Dakar.

## 🚀 Fonctionnalités
- **Recherche d'Hôpitaux** : Filtrez par quartier ou spécialité.
- **Dossiers Détaillés** : Consultez les infos de l'Hôpital Principal, Clinique du Cap, etc.
- **Flux de Réservation** : Choisissez votre médecin et votre créneau en quelques clics.
- **Tableau de Bord** : Gérez vos rendez-vous et votre historique.

## 🛠 Tech Stack
- **React 18** + **Vite**
- **Tailwind CSS** (Styling mobile-first)
- **Lucide React** (Icônes)
- **Framer Motion** (Animations fluides)
- **React Router** (Navigation)

## 📦 Installation & Lancement
1. `npm install`
2. `npm run dev`

## 🎨 Design System
Le design utilise le "Vert Émeraude de Dakar" (#009B77) comme couleur primaire pour une identité visuelle forte et rassurante.
